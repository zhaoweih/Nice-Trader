

#  闲置君 Nice-Trader

## 介绍 Description:

💰闲置君是一个大学校园二手交易APP，可以发布商品，查看商品，评论，实时聊天等功能，同时也具有对学生有利的一些小工具类，例如翻译、查看天气、云笔记等。
